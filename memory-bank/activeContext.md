## Последние обновления (2026-02-06)

### TUI-монитор
- Добавлен `swarm tui` — полноценный TUI на Textual со скроллингом
- Управление: ↑↓ скролл, R обновить, D показать done, Q выход
- Автообновление каждые 2 секунды

### Улучшенные столбцы в мониторе
- **Роль** — для какой роли задача (`target_role`)
- **После** — зависимость (`depends_on`)
- **Назначен** — кому назначена (`target_name`)
- **Работает** — кто выполняет

## Реализованные команды

### Для Лидера (оператора)
- `swarm init` — инициализация среды
- `swarm task add/list` — управление задачами
- `swarm task assign <ID> --agent <имя>` — назначить задачу агенту
- `swarm task close <ID>` — принудительно закрыть задачу
- `swarm agents` — список агентов
- `swarm agents --cleanup` — удалить неактивных агентов
- `swarm monitor` — live-дашборд (4 панели)
- `swarm tui` — TUI-монитор со скроллингом
- `swarm logs` — журнал событий
- `swarm unlock --force` — принудительное снятие блокировок

### Для агентов (все с параметром --agent <имя>)
- `swarm join` — регистрация
- `swarm next --agent имя` — получение задачи
- `swarm lock файл --agent имя` — блокировка файлов
- `swarm done --agent имя` — завершение задачи
- `swarm status --agent имя` — текущий статус
